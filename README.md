# Admin Dashboard

A modern and responsive admin dashboard built with React, Vite, and Tailwind CSS.

## Features

- Responsive Layout
- Sidebar Navigation
- Dashboard Cards
- Charts
- Tables
- Dark Mode
- Modern UI

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

## Installation

```bash
npm install
npm run dev
```

## Project Structure

```
src/
 ├── components
 ├── pages
 ├── layouts
 ├── assets
 └── App.jsx
```
